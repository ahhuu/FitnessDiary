# FitnessDiary (健身日记)

![FitnessDiary Logo](app/src/main/res/mipmap-xxxhdpi/ic_launcher.png)

FitnessDiary 是一款极简、高效的 Android 个人健康管理应用，提供每日打卡、训练计划、饮食记录、身体指标分析、成就系统的一站式服务。

## 🌟 核心功能

- **🚀 每日打卡**：展示本周日历，记录连续坚持天数，追踪今日训练任务。
- **📋 训练计划**：分类管理动作，支持组数/次数设定及图片/视频演示。
- **🥗 智能饮食**：全天候记录餐点，自动计算热量、蛋白质及碳水摄入。
- **📊 身体指标**：可视化 BMI 进度条，动态计算 BMR 与 TDEE 摄入建议。
- **🏆 成就系统**：基于活跃天数的等级进度与勋章解锁。

## 🛠️ 技术架构

- **语言**: Java 
- **架构**: MVVM
- **存储**: Room Database
- **UI**: Material Design 3, ViewBinding, Navigation Component
