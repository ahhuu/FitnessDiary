# FitnessDiary (健身日记)

![FitnessDiary Logo](app/src/main/res/mipmap-xxxhdpi/ic_launcher.png)

FitnessDiary 是一款极简、高效的 Android 个人健康管理应用，提供每日打卡、训练计划、饮食记录、身体指标分析、成就系统与 AI 助手的全方位服务。

---

## 🌟 核心功能

- **🏠 健康仪表盘**：首页右上角健康评分环，五维度（训练/饮食/习惯/身体/坚持）综合评分，阶梯制评分算法更合理；点击查看各维度得分明细。
- **➕ 快捷记录**：底部导航栏中央加号按钮，一键直达智能助理/核心记录/身体指标/生活日常四功能区。
- **🚀 每日打卡**：首页卡片式布局，记录连续坚持天数，追踪今日训练任务与热量消耗；饮食卡片新增脂肪宏量营养素追踪。
- **📋 历史日历**：月度日历历史页面，单元格显示训练标记/饮食热量/步数等多维度数据；日期弹窗查看当日全维度摘要与训练详情，训练详情区分计划目标和当天实际训练，并支持记录仅归属于当天的额外动作。训练计划业务拆分为三个独立页面：计划管理、周/月数据统计、动作库。
- **🥗 智能饮食**：全天候记录餐点，自动计算热量及碳水/蛋白质/脂肪三大宏量营养素；饮食页顶部能量状态横条实时展示摄入 vs 消耗；高频食物 Chips 一键快捷录入；收藏食谱系统支持批量打包添加。
- **👣 步数追踪**：手机传感器自动记录 + 手动修正，支持每日目标配置。
- **😊 情绪记录**：每日 5 档表情选择，轻松记录心情起伏。
- **💧 健康习惯**：喝水、睡眠、用药、体重、围度、便便、经期等多维度习惯追踪，支持 kg/斤 单位切换。
- **⚖️ 我的页面**：v2.4 全面重构 —— 我的状态、荣耀与分析；健身工具箱；我的内容资产；设备与连接（预留）；系统通用设置；关于APP。
- **🏆 成就系统**：21 天打卡挑战、等级进度与勋章解锁，陪伴你持续坚持。
- **🤖 AI 私教**：集成大语言模型，提供智能训练计划制定、饮食分析与进度评估。
- **🎨 深色主题**：支持浅色/深色外观主题切换。
- **🎓 新手引导**：首页五维度评分环及核心卡片引导，帮助新用户快速了解核心功能。
- **🤝 账号与朋友（内测）**：可选邮箱注册、验证与登录；通过好友码建立关系，分享用户逐次确认的训练、打卡和步数摘要，完整健康记录仍只保存在本机。

## 🛠️ 技术架构

### 核心技术栈
- **开发语言**: Java 17 & Kotlin 1.9 (MVVM 架构)
- **本地存储**: Room Database 2.6+ (v30, 25 张数据表)
- **界面设计**: Material Design 3, ViewBinding, Navigation Component (单 Activity 多 Fragment, 4-Tab 底部导航)
- **动画效果**: Lottie Animation
- **AI 服务**: DeepSeek API + 阿里云 DashScope SDK (Qwen 通义千问)

### 关键依赖库
- **数据可视化**: [MPAndroidChart](https://github.com/PhilJay/MPAndroidChart) (体重/睡眠/训练热量趋势统计图表)
- **网络与解析**: OkHttp 4.12 & Gson
- **图片加载**: Glide (含 GIF 动图支持，用于动作库动图预览)
- **条码扫描**: ZXing (扫描食品条形码快速录入)
- **AI 平台**: 阿里云 DashScope SDK (Qwen 通义千问)、DeepSeek API (智能训练计划制定与饮食分析)
- **云端账户与社交**: 腾讯云 CloudBase 身份认证（邮箱验证码）+ PostgreSQL REST/RPC（开发环境小范围内测）

## 🚀 快速上手

### 1. 配置 AI 密钥 (可选)
在项目根目录下的 `local.properties` 文件中配置您的 AI 平台 API Key：
```properties
gemini.api.key=YOUR_GEMINI_KEY
deepseek.api.key=YOUR_DEEPSEEK_KEY
qwen.api.key=YOUR_QWEN_KEY

# CloudBase 账号与朋友功能（可选；留空时本地功能正常运行）
cloudbase.env-id=YOUR_CLOUDBASE_ENV_ID

# Release 签名（仅保存在本机，禁止提交）
signing.store.file=fitnessdiary.jks
signing.store.password=YOUR_STORE_PASSWORD
signing.key.alias=YOUR_KEY_ALIAS
signing.key.password=YOUR_KEY_PASSWORD
```

CloudBase PostgreSQL 的建表、RLS 权限和 RPC 函数脚本位于
[`cloudbase-api/src/main/resources/db/migration/V1__create_social_schema.sql`](cloudbase-api/src/main/resources/db/migration/V1__create_social_schema.sql)。
在 CloudBase SQL 编辑器执行该脚本后，Android 以 CloudBase 登录 JWT 直接调用 PostgreSQL REST/RPC；不需要把数据库账号、密码或管理密钥打包进 APK。开发环境只用于小范围内测，公开运营前需完成套餐、备案与隐私合规评估。

### 2. 构建与运行
```bash
# Windows
gradlew.bat assembleDebug
gradlew.bat installDebug

# macOS / Linux
./gradlew assembleDebug
./gradlew installDebug
```
