# FitnessDiary

![FitnessDiary Logo](app/src/main/res/mipmap-xxxhdpi/ic_launcher.png)

FitnessDiary 是一款基于 Android 平台开发的个人健康管理与饮食记录应用。它旨在帮助用户轻松记录每日饮食、监控身体各项指标，并提供科学的数据分析。

## 🌟 核心功能

### 1. 饮食追踪
- **餐点记录**：支持记录早餐、午餐、晚餐及加餐。
- **详情展示**：自动计算摄入的卡路里、蛋白质、碳水等营养素。
- **食物百科**：内置丰富的食物库，支持分类浏览与关键词搜索。
- **自定义食物**：支持添加个人专属食物到库中。

### 2. 身体指标分析 (BMI/BMR)
- **实时监控**：在个人页面直观显示身高、体重、BMI 及 BMR。
- **深度分析**：
    - **BMI 详情**：通过渐变进度条可视化当前体型分类（偏瘦、正常、偏重、肥胖）。
    - **BMR & TDEE 详情**：根据年龄、性别、身高、体重计算基础代谢，并给出减脂、维持或增肌的每日摄入建议。

### 3. 个人成就系统
- **训练天数**：自动统计开启应用的训练天数。
- **等级系统**：根据活跃天数提升用户等级（从“新手”到“健身达人”）。
- **成就勋章**：解锁特定里程碑（如：第一步、坚持一周、身体管理大师等）。

### 4. 极简 UI 设计
- 采用 **Material Design 3** 设计语言。
- 清新大留白风格，配色清新自然。
- 丰富的 Emoji 图标增强视觉引导。

## 🛠️ 技术栈

- **语言**: Java (完全兼容 Java 8)
- **核心框架**: Android SDK (minSdk 26)
- **架构**: MVVM (Model-View-ViewModel)
- **数据库**: Room Persistence Library
- **UI 绑定**: ViewBinding
- **导航**: Navigation Component
